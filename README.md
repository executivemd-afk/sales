# Sales Check-in

โครงสร้างแอปตัวอย่าง (React + Vite) สำหรับโปรเจค sales-checkin-app

คำสั่งใช้งาน

1. ติดตั้ง dependencies

```bash
npm install
```

2. เริ่มโหมดพัฒนา (dev server)

```bash
npm run dev
```

3. สร้าง build สำหรับ production

```bash
npm run build
```

4. ดูตัวอย่าง build ที่สร้าง

```bash
npm run preview
```

ไฟล์สำคัญที่เพิ่ม:
- src/main.jsx (entry)
- src/App.jsx (component)
- src/index.css (สไตล์)

หมายเหตุ: index.html ถูกตั้งค่าให้โหลด `/src/main.jsx` อยู่แล้ว
